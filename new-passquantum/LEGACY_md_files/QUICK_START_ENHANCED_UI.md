# 🚀 Quick Start Guide - PassQuantum Enhanced UI

## Run the Enhanced Application

```bash
cd /home/lenovo/dev/PassQuantum/new-passquantum/ui
./passquantum-ui
```

That's it! The application will launch with all the enhanced visual effects.

## Troubleshooting

If you encounter any issues:

1. **Rebuild the application**:
   ```bash
   cd /home/lenovo/dev/PassQuantum/new-passquantum/ui
   go build -o passquantum-ui
   ```

2. **Check dependencies**:
   ```bash
   go mod tidy
   ```

3. **Run in debug mode**:
   ```bash
   go run .
   ```
